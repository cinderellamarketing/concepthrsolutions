1

Use DataTags for retained client information

▶

A DataTag is a tag with custom fields attached. The fields only appear when the tag is applied to a contact, keeping things clean for everyone else.

The Retained Client tag has been converted to a DataTag with three fields:

FieldTypeWhat it captures
Renewal DateDateWhen the retainer is due for renewal
PackageDropdownHistoric, Thank You, Session, Bespoke
BreatheHRCheckboxWhether the client uses BreatheHR

To view or edit these fields: open the contact, click Edit, scroll to the Retained Client section, and fill in the details.

Add a description to any custom field that is not obvious. For example, Renewal Date has "When the renewal is due" so future team members understand what goes in each field.
2

Use dropdowns instead of free text wherever possible

▶

Free-text fields are useful when the information is different every time. But if you find yourself typing the same things repeatedly (like contact sources), switch to a single-select dropdown.

Dropdowns let you filter and report accurately. Free text breaks as soon as someone types it differently. One typo ("GRP" vs "grp" vs "GRP network") and Capsule treats them as three different values.

When to use which

Field typeUse when
Single select (dropdown)There is a fixed set of options and you pick one. E.g. Source, Package level
Multi-selectThere is a fixed set of options and you might pick several. E.g. Services offered
Text (free form)The answer is different every time. Always tick "searchable" so you can still find it
DateDates. E.g. Renewal Date, Start Date
CheckboxYes or no. E.g. "Uses BreatheHR"
The Source field on contacts is currently free text. Consider converting it to a single-select dropdown with your main sources (GRP, Referral, Website, LinkedIn, Networking, etc.) so you can report on where contacts come from.
3

Set up tracks for repeatable processes

▶

A track is a sequence of tasks that fires automatically when applied to an opportunity or project. Use them for any process you follow the same way every time.

An Onboarding track has been created on projects as a starting example:

#TaskTiming
1Send welcome emailDay 0 (start)
2Go Cardless / billing setupAfter previous task
3Schedule HR MOTAfter previous task

To apply a track: open a project, click "Add Track" (underneath the stage bar), and select the track.

Key things to know

By default, the next task only appears after the previous one is ticked off. This keeps the list clean but means you need to complete tasks in order.

Always set multi-day gaps to weekdays, not calendar days. Otherwise tasks land on weekends.

Changes made to a track in Account Settings only apply to newly applied tracks. Existing ones stay as they are. To update, delete the track from the project and reapply it.

Once you have your client onboarding process documented outside of Capsule, translate it into a track with specific tasks and day offsets. You can always edit and refine it later.
4

Use task categories to organise your workload

▶

Task categories let you tag each task by the type of work it is. When you are in client work mode, you can filter to only see client work tasks.

Categorising by "Call" or "Email" is not that helpful because a call could be sales, client, or networking. Better to categorise by the type of work:

Examples: Client Work, Sales / BD, Networking, Admin, Marketing

How to use them

1
Go to Calendar and Tasks
Switch to Schedule view (top right).
2
Filter by category
Click the Category filter and select the type of work you want to focus on.
3
Work through the list
Complete tasks, drag and drop to reschedule, tick off when done.

You can rename or delete the default categories (Call, Email, Follow-up) in Account Settings and replace them with ones that match how you actually work.

5

Tasks: schedule view, dates not times

▶

The Calendar and Tasks view is your daily to-do list. Use Schedule view (not Week or Month). It shows tasks as an expandable list, which is much easier to scan than a calendar grid.

Daily routine

1
Open Calendar and Tasks in Schedule view
This shows everything due today and coming up.
2
Work through today's tasks
Click a task to see the linked contact, opportunity, or project.
3
Reschedule anything not done
Drag and drop tasks to the next day or a future date. Do not leave them overdue.

Dates, not times

Do not add times to Capsule tasks. Meetings with specific times belong in your Outlook calendar. Capsule tasks are things you need to get done that day, not at a specific time. Adding times just creates extra admin when things move around.

This is the replacement for the paper and pen. Every enquiry, every follow-up, every action. If it needs doing, it is a Capsule task with a due date. If it is not in Capsule, it will get forgotten.
6

Transpond: managing contacts and costs

▶

Transpond charges based on the number of active contacts, not the total in Capsule. The brackets are: up to 250 (free), up to 1,000, then above 1,000.

If you are only marketing to 200 people, you should not be paying for the 1,000+ bracket. Mark contacts you are not actively emailing as Inactive in Transpond. They stay in Capsule but do not count towards your Transpond subscription.

When someone unsubscribes

They are automatically marked as unsubscribed and will not receive any more emails. The contact stays in Capsule and nothing is deleted.

To manually change a contact's Transpond status: open the contact, find the Transpond section on the right-hand side, click the dropdown, and set to Active, Inactive, or Unsubscribed.
